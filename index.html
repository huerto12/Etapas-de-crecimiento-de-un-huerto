La elaboración de un huerto dentro de las instalaciones está pensado para que los alumnos puedan acceder a una alimentación balanceada, que puedan consumir alimentos frescos y puedan aprender conceptos de jardinería. Para ello, se pensó en una página que calcule el crecimiento y etapa de las plantas según la temperatura diaria y el tipo de planta, para posteriormente hacer diferentes pedicciones.
